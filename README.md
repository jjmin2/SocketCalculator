# SocketCalculator (간단한 산술연산 인터넷 계산기)

TCP 소켓 통신을 기반으로 클라이언트가 서버에게 사칙연산을 요청하고 서버가 연산결과를 
Response 객체로 반환하는 구조의 인터넷 계산기 프로그램입니다.

---

## 주요 기능

- plus / minus / times / division 사칙연산 지원
- 클라이언트가 서버에게 텍스트 명령 전송
- 서버에서 결과/오류/종료 상태를 Response 형식으로 전송
- 'break' 입력 시 연결 종료
- 서버는 다중 클라이언트 접속을 스레드 풀로 처리

---

## 프로그램 구성 요소

| 구성 | 설명 |
|-----|-----|
| `Server` | 7777 포트에서 클라이언트 연결 대기, 요청마다 스레드 생성 |
| `Capitalizer (Thread)` | 클라이언트 요청을 실제 연산 처리하는 스레드 |
| `Client` | 사용자 입력을 서버로 전송하고 결과를 화면에 출력 |
| `Response` | 서버-클라이언트 간 메시지 구조 (code, value, message) |
